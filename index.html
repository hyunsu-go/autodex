Overview

Abstract

Real dexterous-grasp data is expensive: simulation is scalable but unreliable for contact, while human teleoperation is physically valid but slow and biased by operator choices. AutoDex removes the human from the per-trial loop entirely. It is an automated real-world data-collection system that proposes geometrically feasible grasps in simulation and validates each one by physically executing it with a multi-fingered hand.

The system closes the full loop without intervention: it accepts candidates from a generator, localizes the object with dense 20-camera tracking, plans and executes collision-monitored motions, labels lift-and-hold success or failure, and actively resets the object for the next trial — then repeats. Using AutoDex we collect 3,593 real-world-validated grasps (successes and failures) across the Allegro and Inspire hands on 100 objects, 4.8× faster than teleoperation; real-world validation raises downstream grasp success from 34% to 76%.

The autonomous loop

The cycle it repeats by itself

Every trial runs these four steps with no human at any of them — then the loop closes and starts the next grasp. Click a step.

4 Reset  ↻

1 · Perceive

A dense 20-camera rig localizes the object — even under severe hand–object occlusion — and reconstructs the surrounding scene, so the system knows exactly what it is grasping and where.

2 · Plan & grasp

A candidate grasp is selected and a collision-free arm trajectory is planned to it. The hand closes on the object — with torque monitoring guarding every contact so the robot can run unsupervised.

3 · Lift & label

The robot lifts and holds. Lift-and-hold success or failure is measured automatically and written as a ground-truth label — the dataset records both the wins and the misses.

4 · Reset ↻

The object is rarely where the next trial needs it. AutoDex re-grasps and re-poses it to the next target pose by itself — the step that lets the loop never stop. Then it begins again.

The hard part

Why it can run unattended: it resets itself

Every other real-world pipeline stops when a human has to re-place the object. AutoDex doesn’t. When a trial ends, the robot re-grasps the object and actively places it into the next pose — even pose transitions that passive dropping can almost never reach.

No human re-placement. Active reset is what turns a sequence of single grasps into hours of continuous, unsupervised collection.

It also keeps coverage broad: the robot can drive the object into specific, otherwise-unreachable orientations, so the dataset isn’t biased toward whatever pose the object happens to settle in.
